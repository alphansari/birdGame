# Bird Game

A simple **Flappy Bird clone** built in **Unity**. 

---

## Gameplay
Press **Space** to flap your wings and avoid the pipes!  
The game ends when you hit a pipe or the ground.

---

## Features
- Smooth physics-based movement  
- Score counter  
- Game over screen  
- Background music and flap sound effect  
- 2D art and animations

---

##  How to Play
1. Open the project in **Unity Editor (2022 or newer)**.  
2. Run the scene called `MainScene` (or your main scene name).  
3. Press **Space** to start flying.

---

## Technologies Used
- Unity Game Engine  
- C# Scripts  
- Audio system with `AudioSource`  
- Simple 2D sprites and colliders
